# Swift로 숫자야구 게임 만들기

## 게임 룰 설명
⚾ 숫자야구 게임은 **두 명이 플레이하는 추리 게임**으로, **상대방이 설정한 3자리의 숫자를 맞추면 끝이 납니다.**

각 자리의 숫자와 위치가 맞으면 스트라이크, 숫자만 맞고 위치가 다르면 볼로 판정됩니다.

**투수(컴퓨터)** 는 투구전략 ```answer```를 생각해놓았고, **타자(사용자)** 가 투구전략 의도대로 3스트라이크를 기록하면 삼진아웃을 당하며 게임이 종료됩니다!
