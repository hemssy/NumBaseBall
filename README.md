# Swift로 숫자 야구 게임 만들기

⚾️ 숫자 야구 게임은 **두 명이 플레이하는 추리 게임**으로, **상대방이 설정한 3자리의 숫자를 맞추면 끝이 납니다.**

각 자리의 숫자와 위치가 모두 맞으면 스트라이크, 숫자만 맞고 위치가 다르면 볼로 판정합니다.
스트라이크가 3일 때 게임이 종료됩니다.
